## mixins & plugins

| Feature     | Mixin                                     | Plugin                                         |
| ----------- | ----------------------------------------- | ---------------------------------------------- |
| Purpose     | Code reuse and composition within an app. | Extending functionality of a host application. |
| Integration | Direct merging into a class.              | Interacting through defined extension points.  |
| Scope       | Internal code organization.               | External extensibility and customization.      |
| Nature      | A set of reusable functionalities.        | A modular, add-on component.                   |

The part of a transmitted message that contains the actual data intended for the recipient, excluding headers or metadata.

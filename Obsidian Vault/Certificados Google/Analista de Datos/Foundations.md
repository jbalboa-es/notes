# Introducing data analytics and analytical thinking
## Six Steps of Data Analysis

- Ask
- Prepare
- Process
- Analyze
- Share
- Act
## Data ecosystem

Consists of various elements that interact to produce, manage, store, analyze and share data, including hardware, software, and users.

## Cloud serves

Cloud serves as a virtual location for data storage

## Common Misconceptions

- Data Scientists: Create new questions using data
- Data Analysts: Find answers to existing questions

- Data analysis: Involves collecting and organizing data to draw conclusions.
- Data analytics: Is a broader concept encompassing all aspects of data management and usage.

## The Role of Human Insight

Combining data with human experience and insights from subject matter-experts enhances decision-making
## Analytical thinking

The process of identifying and defining a problem, then solving it by using data in an organized, step-by-step manner
### Key aspects

- Visualization
- Strategy: A strategic mindset helps data analysts focus on goals and improve the quality of data collected
- Problem-Orientation: Keep the problem in mind.
- Correlation: Identifying relationships between data points. Remember that correlation does not imply causation
- Big-Picture Thinking: Seeing the overall context and opportunities
- Detailed-Oriented Thinking: Focusing on the specifics that contribute to executing a plan.

Versatile thinking is emphasized, allowing individuals to develop skills in analytical, creative and critical thinking
### Techniques for Problem Solving

- Five Whys: To identify the root cause.
- Gap analysis: To evaluate current processes and identify areas for improvement.

### Data-driven decision-making

- Using facts to guide business strategies, leading to better insights and informed choices.
### Key Analytical Skills for Data-Driven Decisions

**Analytical skills:** Qualities and characteristics associated with using facts to solve problems

- Curiosity and Context: Develop curiosity about data, recognizing patterns and relationships to make predictions and draw conclusions.
- Technical mindset: Build on gut feelings by seeking facts and using analysis to inform decisions. Allows to break down tasks into smaller, manageable steps.
- Data Design: Organizing data logically.
- Data Strategy: Incorporates people, processes, and tools to solve problems.
